# Logistics-warehouse-Dataset
一个简单的物流仓库的数据集
图片从百度图片和谷歌图片上进行爬取
数据标签由labelme进行标注，labelme的github链接：https://github.com/wkentaro/labelme
由于数据集是用来做语义分割的，因此保存的格式为json格式，主要保存了points的坐标，标注的内容主要是货架、AGV小车等仓库常见物体
